gcc -c arquivo.c      -> Criar o arquivo obj

ar -rc libarquivo.a arquivo.o      -> Criar a biblioteca estatica
                                   -> -r   =  substituir algum arquivo velho, caso tenha
                                   ->  c   =  criar a biblioteca de fato
                                   ->  libarquivo.a  = nome na biblioteca, esse lib é padrão
                                   ->  arquivo.o   = informando que é para criar a partir do arquivo.o

ranlib libarquivo.a   -> Criando um índice para essa biblioteca
                      -> Definindo o cabeçalho com os símbolos do arquivo.o
                      -> Isso ajuda a temos uma rápida referência para essa biblioteca na hora de compilar

ar -t libarquivo.a    -> informa o conteúdo da biblioteca que é o arquivo.o

gcc main.c -L. -larquivo -o main.exe    -> Criando o executável com a 'linkagem' da biblioteca
                                        -> -L. = informando que é para procurar essa biblioteca no dir local
                                        -> -larquivo = informando qual biblioteca é
                                        -> É importante usar o lib na criação da biblioteca, pois quando formos
                                        -> fazer a 'likagem', basta informa esse -l que o commpilador entende
                                        -> como lib
                                        -> -o main.exe = padrão do gcc para criar o executável

Se colocarmos a biblioteca no local padrão que o compilador procurar e colocarmos
o arquivo.h no local padrão também.
Basta executarmos o comando: gcc main.c -larquivo -o main.exe -> Que criamos o executável
